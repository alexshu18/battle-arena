# Battle-arena
This is battle arena. It was developed on Java, and having all basic concepts of OOP.
It is still in progress, but you can clone the repo and run Main.java to test it. 
You can choose whether you play as a Warrior, or Archer. Different units - different damage. Also in plan that you will have a loot after defeating the enemy, and also you can up your level, which will up your hp and damage.
